* USUARIOLOCAL: Usuario que has creado para el ENV de ODOO
* generalmente es 1001 a sabiendas que el primer usuario que has creado es pepe, jesus, etc.... 
* Se puede ver el UID haciendo un cat /etc/passwd | grep <USUARIOLOCAL> y el GID en cat /etc/group  | grep <USUARIOLOCAL>
